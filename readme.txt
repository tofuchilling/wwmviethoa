ĐỐI VỚI BẢN CLIENT:
- Vào thư mục cài game, ví dụ D:\wwm\wwm_standard(bản standard) hoặc wwm_lite(bản lite)\LocalData\
- Copy thư mục Patch và chép đè vào

ĐỐI VỚI BẢN STEAM:
- Vào thư mục Steam\steamapps\common\Where Winds Meet\
- Copy thư mục Package và chép đè vào

** Vào game chọn ngôn ngữ đầu tiên (gốc là tiếng Anh) 
** Lưu ý: một số đoạn text sẽ không dịch được tiếng Việt (mô tả tâm pháp, bát úp, ...) do text tải từ server của NetEase
** Vì đây chỉ can thiệp vào file bin ngôn ngữ, không phải các file resource, do đó đảm bảo không bị ban acc.

*** Đối với những ai muốn đổi qua lại giữa tiếng Anh và tiếng Việt:
    1. Chép lại file gốc translate_words_map_en vào Package\HD\oversea\locale
    2. Đổi tên file translate_words_map_en trong thư mục Việt hóa thành translate_words_map_de
    3. Copy file translate_words_map_de đè vào thư mục Package\HD\oversea\locale
    4. Vào game đổi sang ngôn ngữ cuối cùng (gốc tiếng Đức)
